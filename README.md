# Projet API Artiste

Ce projet est une API développée en Go (Golang) permettant de gérer et de filtrer des informations sur des artistes. L'API récupère des données externes sur les artistes et leurs relations, puis les met à disposition pour l'affichage et le filtrage sur une page web.

## Fonctionnalités

- **Récupération des données** : Les informations sur les artistes et leurs relations sont récupérées depuis des API externes.
- **Filtrage** : Les utilisateurs peuvent rechercher des artistes par nom, filtrer par membre, trier par date de création ou par premier album, et sélectionner des artistes selon des critères de localisation et de nombre de membres.
- **Affichage Web** : Les données sont affichées via un modèle HTML pour une visualisation directe.
